# 🎯 AI Powered Placement Preparation Assistant

📌 Overview

The AI Powered Placement Preparation Assistant is a Machine Learning and Streamlit-based web application that helps students prepare for placements by providing:

🎯 Career Domain Recommendation
📊 Placement Probability Prediction
💻 Interactive and Professional User Interface
🔐 Login & Registration System

This project combines Machine Learning with a modern web interface to assist students in making informed career decisions.

✨ Features
🔐 User Login & Registration
🎯 Career Domain Prediction using Machine Learning
📊 Placement Prediction using Deep Learning (TensorFlow)
📚 Required Skills Recommendation
🎨 Modern Responsive UI
🌙 Professional Dashboard Design
⚡ Fast Predictions


🛠 Technologies Used
Python
Streamlit
TensorFlow / Keras
Scikit-learn
NumPy
Pandas
Joblib
Streamlit Option Menu
